# LangGraph ReAct Agent Template


## Installation avec uv

```sh
git clone https://github.com/Montbel/TOULON-CHRISTIAN-formation-langchain-lbke-examen-cpf
cd ./https://github.com/Montbel/TOULON-CHRISTIAN-formation-langchain-lbke-examen-cpf

# ==================================================
# Create venv
python -m venv venv

# ==================================================
# Activate venv
source nom_du_venv/bin/activate

# ==================================================
# Install dependencies
uv sync

# ==================================================
# Launch langgraph
uv run langgraph dev
